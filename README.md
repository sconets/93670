　　研究团队之后通过组织多台国际设备天地协同观测，综合射电干涉阵列、光学、红外望远镜以及空间高能天文台的数据，将FRB 20190520B定位于一个距离地球30亿光年的贫金属的矮星系，确认其近源区域拥有目前已知的最大电子密度，并发现迄今第二个快速射电暴持续射电源对应体（PRS）。
